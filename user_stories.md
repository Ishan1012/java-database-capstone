# User Stories - Patient Appointment Portal

## Admin Stories
- As an admin, I want to manage user accounts so that I can control access to the system.
- As an admin, I want to view system logs to ensure functionality is stable.

## Patient Stories
- As a patient, I want to book an appointment so that I can see a doctor.
- As a patient, I want to view my appointment history to track my health visits.

## Doctor Stories
- As a doctor, I want to set my availability so that patients know when I am free.
- As a doctor, I want to view my daily schedule to prepare for my patients.
